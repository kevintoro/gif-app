# GIF Search App 🎯

A React application that allows users to search and discover GIFs using the GIPHY API.

## Features ✨

- Search for GIFs by keywords
- View previous searches
- Click on previous search terms to quickly search again
- Responsive grid layout of GIF results
- Clean and intuitive user interface

## Tech Stack 🛠️

- React
- TypeScript
- Vite
- GIPHY API
